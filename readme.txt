Requires Woocommerce. Adds an option to assign a role to user when a specific product has been bought

== Description ==

Creates a metabox to woocommerce product edit pages with an option to assign a role to user when the product is bought. 

== Installation ==

1. Upload the extracted zip to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. When editing a product, add a role in the PurchaseRoles -metabox.
